CSV Import

Daten aus deinen Apps importieren

Polar Flow
Adidas Running
Alpha Progression
Export aus Polar Flow
1. Öffne flow.polar.com im Browser
2. Klick auf eine Aktivität → Export → CSV
3. Oder: Profil → Daten exportieren (alle Aktivitäten)
Spalten: Date, Distance (km), Duration, Average heart rate
📂
Polar CSV auswählen oder ablegen
Spalten werden automatisch erkannt
Export aus Adidas Running
1. Öffne die Adidas Running App
2. Profil → Einstellungen → Daten exportieren
3. Du bekommst eine E-Mail mit ZIP → CSV entpacken
Spalten: date, distance (m), duration (s) — wird automatisch umgerechnet
📂
Adidas Running CSV auswählen
Meter → km und Sekunden → Pace werden umgerechnet
Export aus Alpha Progression
1. Öffne Alpha Progression
2. Profil (unten rechts) → Export → Workouts als CSV
Spalten: Date, Exercise, Sets, Reps, Weight — Volumen wird berechnet
📂
Alpha Progression CSV auswählen
Sessions werden nach Datum gruppiert
